WebFrix

The event is meant for people interested in web desgning who will be given the opportunity to create a small website during a given time and present it.

Rules:

  • Use of vanilla HTML/CSS for web design is allowed
  • Use of the Bootstrap framework is encouraged, though not mandatory
  • Use of Bootstrap templates is strictly discouaraged and will result in negative evaluation
  • Use of any other frontend web framework apart from Bootstrap is also not allowed
  • Use of any dynamic server side scripting language like PHP is prohibited since, it is only a frontend design contest
  • Your website will be judged on the basis of the responsiveness and mobile adaptability it provides
  • A minimum of 5 web pages must be created including the homepage

Overview:

  • Create a website with atleast 4 stories on any trending topics of your choice.
  • The homepage must showcase any 2 top trending stories with options for Read More.
  • You can also use dummy stories on the main page that do not redirects to other internet websites, but 4 pages or more stories in site is mandatory.
  • You'll be required to upload the website as a zip file and upload it to our network. Instructions for the same will be provided to you.

References and Other Guidelines:

  • You can use Mashable India website for your reference, but you are requested to put in your custom inputs for alignment with appropriate evaluation measures.
  • You can also use topics from Twitter's trending section.
  • These references can also be used for Bootstrap and CSS3:
    • Twitter Bootstrap Succinctly Ebook on Syncfusion
    • CSS3 Succinctly Ebook on Syncfusion

Each participant will have to present his/her website (maximum: 2 minutes) which will be crucial for the evaluation process.

Evaluation criteria:

  1. Singularity in design across pages
  2. Responsiveness: (website will be tested on the following devices)
    1. Desktop Web Browser Google Chrome/Mozilla Firefox (1920x1080px)
    2. iPhone 5S Safari/Chrome
    3. Android Device Chrome (regular phone screen size)
  3. User appealing homepage
  4. Navigation bar and page linking
  5. Presentation
